# Task 1: ATM Cash Withdrawal System

## Overview
This project models a standard ATM withdrawal process. It includes secure PIN authentication, account balance checks, and transaction processing for cash withdrawals or deposits.

## Features
- **PIN Verification:** A security layer that validates the user's personal identification number.
- **Menu Options:** Navigation for checking balance, depositing funds, or withdrawing cash.
- **Transaction Logic:** Real-time updates to the account balance based on user actions.
- **Error Handling:** Feedback for incorrect PIN entries or insufficient funds during withdrawal.

## File Structure
- `pseudocode.md`: Logical steps of the ATM operations in plain English.
- `flowchart.dot`: Graphviz source code for the ATM algorithm.
- `flowchart dot.png`: Visual diagram generated from the DOT code.
- `flowchart.mmd`: Mermaid.js source code for the ATM algorithm.
- `flowchart mermaid.png`: Visual diagram generated from the Mermaid code.
- `llm-conversations.txt`: Recorded interaction with the AI assistant for this task.
