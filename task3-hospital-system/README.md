# Task 3: Hospital System

## Overview
This project simulates a Hospital Management and Appointment System. It focuses on patient registration, medical department selection, and real-time doctor availability checks to ensure a smooth booking process.

## Features
- **Patient Authentication:** Handles new registrations and existing patient logins.
- **Department Selection:** Allows users to choose between Cardiology, Pediatrics, and General Surgery.
- **Availability Check:** Validates if a doctor is free before confirming the appointment.
- **Slot Booking:** Assigns specific time slots for successful reservations.

## File Structure
- `pseudocode.md`: Logic description in plain English.
- `flowchart.dot`: Graphviz source code.
- `flowchart dot .png`: Visualization of the Graphviz flowchart.
- `flowchart.mmd`: Mermaid source code.
- `flowchart.mmd.png`: Visualization of the Mermaid flowchart.
