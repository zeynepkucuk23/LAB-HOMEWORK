# Algorithms Flowcharts

This repository contains flowchart and pseudocode solutions for algorithm modeling assignments.

## Tasks Included:
- **Task 1** – ATM Cash Withdrawal System
- **Task 2** – Online Shopping Cart System
- **Task 3** – Hospital Appointment System
- **Task 4** – University Course Registration System

## Each Task Folder Includes:
- **pseudocode.md**: Step-by-step logic.
- **flowchart.dot**: Graphviz source code.
- **flowchart dot.png**: Exported diagram.
- **flowchart.mmd**: Mermaid source code.
- **flowchart mermaid.png**: Exported diagram.
- **README.md**: System description.
- **llm-conversations.txt**: Interaction logs.
