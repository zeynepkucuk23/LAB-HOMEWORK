# Task 4: Course Registration System

## Overview
This system simulates an academic course registration process. It ensures that students are authenticated before choosing courses and checks for seat availability (quota) before finalizing any enrollment.

## Features
- **Student Authentication:** Secure login for students to access the registration portal.
- **Course Selection:** Students can view and select from a list of available academic courses.
- **Quota Validation:** An automatic check is performed to prevent enrollment if a course is full.
- **System Feedback:** Real-time messages are provided for successful registrations or quota errors.

## File Structure
- `pseudocode.md`: Step-by-step logic of the registration process in plain English.
- `flowchart.dot`: Graphviz source code for the system flowchart.
- `flowchart dot.png`: Visual representation generated via Graphviz.
- `flowchart.mmd`: Mermaid source code for the system flowchart.
- `flowchart mermaid.png`: Visual representation generated via Mermaid.
- `llm-conversations.txt`: Logs of the interaction with the AI assistant during development.
