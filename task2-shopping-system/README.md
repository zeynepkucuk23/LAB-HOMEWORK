# Task 2: Online Shopping Cart System

## Overview
This project simulates a basic online shopping experience. The algorithm manages product selection, maintains a virtual shopping cart, and calculates the final total with applicable discounts. It demonstrates loops for item entry and conditional logic for price reductions.

## Features
- **Product Listing:** Displays available items and their prices to the user.
- **Cart Accumulation:** Uses a loop to allow users to add multiple products until they choose to checkout.
- **Discount Mechanism:** Automatically applies a 10% discount if the total cart value exceeds 1000 TL.
- **Final Receipt:** Provides a summary of the total amount to be paid after all logic is processed.

## File Structure
- `pseudocode.md`: Step-by-step logic of the shopping process in plain English.
- `flowchart.dot`: Graphviz source code for the shopping system flowchart.
- `flowchart dot.png`: Visual representation generated via Graphviz.
- `flowchart.mmd`: Mermaid source code for the shopping system flowchart.
- `flowchart mermaid.png`: Visual representation generated via Mermaid.
- `README.md`: Task-specific documentation (this file).
- `llm-conversations.txt`: Logs of the interaction with the AI assistant.
